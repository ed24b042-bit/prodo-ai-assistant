# ProDo Frontend ⚡

This is the Next.js frontend application for the ProDo Conversational AI Executive Assistant.

## Learn More
For complete details on features, architecture, database schemas, and local setup instructions, please refer to the main [README.md](../README.md) in the root of the workspace.

## Getting Started
First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
